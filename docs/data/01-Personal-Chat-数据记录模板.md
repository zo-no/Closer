## 使用说明

- 一人一条主记录
- 每次关键互动后更新一次
- 先记录事实，再写判断
- 这份模板优先服务个人执行，不追求复杂统计

## 样本主表

| 字段 | 填写说明 | 示例 |
| --- | --- | --- |
| 样本编号 | 统一编号，便于复盘引用 | PC-001 |
| 对象代号 | 不写真实姓名，可匿名 | A01 |
| 来源渠道 | 社交软件 / 朋友介绍 / 线下添加 / 其他 | 社交软件 |
| 首次建立联系日期 | 第一次开始互动的日期 | 2026-04-08 |
| 当前阶段 | 新接触 / 互动中 / 已邀约 / 已确认见面 / 暂缓 / 结束 | 互动中 |
| 对方画像倾向 | 高意愿但低方法 / 方法焦虑认真型 / 未知 | 未知 |
| 主动性评分 | 1-5 | 3 |
| 舒适度评分 | 1-5 | 4 |
| 现实可推进性 | 1-5，是否适合进入线下 | 3 |
| 是否发出邀约 | 是 / 否 | 否 |
| 邀约结果 | 接受 / 未定 / 拒绝 / 未邀约 | 未邀约 |
| 当前结论 | A 已确认首次见面 / B 保持轻联系 / C 明确不推进 / D 信息不足等待触发 | D |
| 下一步动作 | 下一次应该做什么 | 2 天后轻量重启 |

## 互动日志

每次关键互动追加一条：

| 日期 | 动作类型 | 我发了什么 | 对方怎么回 | 判断 | 下一步 |
| --- | --- | --- | --- | --- | --- |
| 2026-04-08 | 破冰 | 你今天像刚下班还是刚打完仗？ | 哈哈刚下班 | 能接球，可继续 | 追问轻连接 |

## 周度汇总

每周更新一次：

| 周期 | 新增样本数 | 有效互动数 | 发出邀约数 | 接受邀约数 | 已确认见面数 | 主要卡点 |
| --- | --- | --- | --- | --- | --- | --- |
| 2026-W15 | 0 | 0 | 0 | 0 | 0 | 待开始 |

## 快速判断规则

- `主动性 <= 2` 且 `舒适度 <= 2`：降频或结束
- `主动性 >= 3` 且 `舒适度 >= 3`：可继续推进
- `主动性 >= 3` 且 `现实可推进性 >= 3`：可评估邀约
- 连续 2 次主动发起未获有效回应：结束本轮

## 备注规范

- 用事实语言，不写模糊判断
- 少写“感觉她还行”，多写“愿意展开回答并反问”
- 若出现拒绝或失联，写清发生节点
